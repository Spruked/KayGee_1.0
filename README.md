# KayGee 1.0: Bio-Inspired Cognitive Dashboard

KayGee 1.0 is a next-generation, bio-inspired cognitive presence and dashboard. It integrates:
- **Cognitive Resonance**: Real-time phase coherence and resonance metrics
- **Symbolic Cognition**: 3D space field visualizer (complex phasor math)
- **Audio System**: (Planned) Cochlear and POM voice modules
- **Philosophical Balance**: Dynamic metrics inspired by Kant, Hume, Locke, Spinoza
- **Technical Metrics**: Deep system state, integrity, and live conversation

## Features
- **Live 3D Space Field**: Visualizes cognitive geometry in real time
- **Resonance Meter**: Tracks phase coherence and dominant frequency
- **System State**: Confidence, stability, philosopher mode, response time
- **Audio System**: (Planned) TTS, waveform, mic input, corrections
- **Philosophical Balance**: Visualizes cognitive influences
- **Integrity**: Drift detection and system hash

## Quick Start
1. **Install dependencies**
   ```sh
   cd Kay_Gee_1.0/frontend
   npm install
   ```
2. **Run the dashboard**
   ```sh
   npm run dev
   ```
3. **(Optional) Start backend**
   ```sh
   cd ../backend
   uvicorn main:app --reload
   ```

## Project Structure
- `Kay_Gee_1.0/frontend/` — React/Vite/Three.js dashboard
- `Kay_Gee_1.0/backend/` — FastAPI backend (planned: live data, audio, websocket)
- `visualization/` — Space field generator (Python)
